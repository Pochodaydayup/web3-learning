# 第 7 章练习题

## 概念题

1. 为什么“L2 更便宜”这句话在工程上过于粗糙？
2. 为什么 calldata 成本在很多 Rollup 环境里会成为关键？
3. L2 费用结构和 L1 费用结构最核心的差异是什么？
4. 为什么现代链上优化要把数据结构和费用结构一起看？

## 分析题

1. 某团队在 L2 上只盯着 opcode 优化，却忽略了大量重复数据发布。请分析为什么收益可能有限。
2. 一个批处理系统执行逻辑不复杂，但费用依然偏高。请从数据发布角度分析原因。
3. 某前端交互在单个用户下很便宜，但规模上来后成本上升明显。请分析可能和哪些数据因素有关。

## 代码题

1. 为一个 L2 交互路径画成本结构图，区分本地执行与数据发布。
2. 审查一个你熟悉的参数结构，分析它是否存在可压缩空间。
3. 写一份“L1 思维 vs L2 思维”的成本判断对照表。
