# 第 5 章练习题

## 概念题

1. 函数选择器为什么只看函数名和参数类型，而不看返回值？
2. 部署字节码和运行时代码分别在做什么？
3. 为什么说合约调用进入后通常先经历 selector dispatch？
4. ABI 与 selector dispatch 在职责上有什么区别？

## 分析题

1. 为什么 fallback / proxy / selector 碰撞这几个话题很容易互相关联？
2. 如果你只知道 ABI，不理解运行时代码分发，会在哪些底层场景里感到吃力？
3. 为什么反编译输出里常常会出现一堆 selector 比较分支？

## 代码题

1. 为一个简单合约计算几个函数的 selector，并解释差异来源。
2. 编译一个最小合约，区分其部署字节码与运行时代码。
3. 画出“函数签名 -> selector -> calldata -> dispatch -> 目标逻辑”的流程图。
