# 第 8 章练习题

## 概念题

1. `address.code.length` 真正回答的是什么问题？
2. 为什么“当前无代码”不等于“它就是稳定的 EOA 身份”？
3. 构造器阶段为什么会让 `isContract` 一类逻辑失效？
4. 为什么现代 Ethereum 中不应再把 `SELFDESTRUCT` 当作传统删除工具？

## 分析题

1. 某协议用“禁止合约地址参与”来防机器人。请分析这类设计的边界和漏洞。
2. 某系统读到代理地址上有代码，就直接把它当成完整业务实现来理解。请分析这种源码阅读误区。
3. 某老项目文档里写“未来可以用 `SELFDESTRUCT` 完成彻底清理”。请分析这在现代语义下的问题。

## 代码题

1. 写一个最小辅助函数，分别输出某地址的 `code.length` 与 `codehash` 观察结果，并解释它们各自的限制。
2. 检查你熟悉的一个协议地址，判断它是代理、实现还是普通逻辑合约。
3. 为一个“禁止合约调用”的限制逻辑写一段安全评审意见。
