# 第 8 章延伸阅读

## 必读

- JSON-RPC 中 `eth_call`、`eth_estimateGas` 的说明
- 你正在使用的客户端库关于 simulation / static call / gas estimate 的文档
- 目标协议关于 slippage、deadline、allowance 或 execution risk 的交互说明

## 进阶

- mempool 排队、价格变化和交易竞争对 simulation 的影响资料
- 交易失败分类、revert data 解析和 receipt 调试资料
- 多步骤交易、批量调用和复杂路由下的前置检查设计文章

## 实战

- 给你的 dApp 整理一份“发送前能确定什么，不能确定什么”的说明
- 对比两个主流 dApp 的交易前提示文案
- 复盘一次“模拟成功但真实失败”的案例，并写出根因分析
