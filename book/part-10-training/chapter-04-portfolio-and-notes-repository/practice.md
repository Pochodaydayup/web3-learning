# 第 4 章项目实践

## 实践目标

这次实践要把你已经拥有的项目和笔记，真正整理成一个有导航、有层次、可长期维护的公开学习资产入口。

## 操作步骤

1. 选择一个现有 GitHub 仓库，或者新建一个专门的索引仓库。
2. 设计首页结构，至少包含：
   - 学习主线
   - 代表项目
   - 源码阅读笔记
   - 面试与训练输出
3. 给每个代表项目补上证据链入口。
4. 给至少一篇源码阅读笔记补上固定模板。
5. 最后写一页“未来三个月如何持续维护这个仓库”的节奏计划。

## 交付物

你应该交付：

1. 一个整理后的索引页
2. 一份源码笔记模板
3. 一份未来维护计划
