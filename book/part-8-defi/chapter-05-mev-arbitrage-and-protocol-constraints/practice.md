# 第 5 章项目实践

## 实践目标

这次实践要让你从“知道 MEV 存在”走到“能用它审视协议路径”。你需要围绕一个真实操作，分析公开博弈会怎样改变结果。

## 操作步骤

1. 选择一个场景：
   - 大额 swap
   - 清算
   - 铸造 / 赎回
   - 奖励领取
2. 写出理想情况下用户以为会发生什么。
3. 再写出公开链环境下可能出现的额外参与者和行为。
4. 分析：
   - 用户是否可能被夹击
   - 是否存在明显套利路径
   - 参数能否减轻问题
5. 最后提出 2 到 3 条设计改进建议。

## 交付物

你应该交付：

1. 一份路径博弈分析
2. 一张参与者关系图
3. 一份设计约束与改进建议清单
