# 第 3 章延伸阅读

## 必读

- Uniswap V2 官方文档与 core 仓库 README
- Uniswap V2 Pair、Factory、Router 的核心源码
- 讲解恒定乘积 AMM 与 LP 份额机制的经典文章

## 进阶

- 对 Uniswap V2 预言机、累计价格和 TWAP 机制的技术解读
- 审计报告或高质量源码解析文章，观察专业审计视角如何拆解协议
- 与 SushiSwap、PancakeSwap 等 V2 体系分叉实现做对比阅读

## 实战

- 在自己的源码笔记仓库里建立一份 Uniswap V2 阅读专题，按“结构 / 状态 / 入口 / 风险 / 问题”分栏整理
- 选一个真实 swap 交易，对照链上事件与源码路径走一遍
- 尝试实现一个最小 Pair 原型，只保留最核心的状态和检查逻辑
