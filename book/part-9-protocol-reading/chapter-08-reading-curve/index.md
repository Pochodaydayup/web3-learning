# 第 8 章：阅读 Curve

前面已经建立了读协议的方法，也读过 OpenZeppelin、Uniswap V2、Aave、EigenLayer、Uniswap V3 和 MakerDAO。接下来很自然的一站，就是 Curve 以及它背后的稳定交换思路。它很适合训练你从“普通 AMM”切换到“带假设的定制流动性数学”。

## 本章要解决什么

本章要解决的是“为什么稳定资产池不能只用普通恒定乘积来理解；为什么 Curve 的价值在于把资产相关性和滑点目标写进数学设计；为什么阅读这类协议时必须同时盯着 invariant、参数、费用路径和治理控制面”这个问题。

## 读完后你应该会什么

- 知道阅读稳定交换系统时该先抓什么主线
- 看懂 amplification、fee、virtual price 等概念在系统中的位置
- 区分普通 AMM 阅读法和稳定资产池阅读法
- 写出一份结构清楚的 Curve 源码笔记

## 本章核心概念

- **放大系数（Amplification Coefficient）**：Curve 类稳定曲线用来调整价格敏感度的关键参数。
- **不变量曲线（Invariant Curve）**：协议用来约束池子状态和报价关系的核心数学曲线。
- **Virtual Price**：稳定池份额对底层资产价值的内部参考价格。
- **Imbalance**：池子不同资产储备相对目标配比发生偏离的状态。
