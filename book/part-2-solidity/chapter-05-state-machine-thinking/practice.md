# 第 5 章项目实践

## 实践目标

这次实践要让你亲手把一个小合约重写成更清晰的生命周期系统。重点不是增加功能，而是提升结构表达力。

## 操作步骤

1. 选择一个你已经写过的合约，例如 Bank、Voting、Auction、Staking。
2. 先列出它现在有哪些隐含阶段。
3. 画一张状态转移图。
4. 再回到代码，标记：
   - 哪些函数依赖阶段
   - 哪些检查目前不够显式
   - 哪些动作本应不可逆
5. 最后写一页重构建议，说明如果重写，你会如何表达它的生命周期。

## 交付物

你应该交付：

1. 一张状态转移图
2. 一份合约生命周期分析笔记
3. 一份重构方案说明
