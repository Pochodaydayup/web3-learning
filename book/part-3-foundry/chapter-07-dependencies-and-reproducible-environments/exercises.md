# 第 7 章练习题

## 概念题

1. 为什么可复现环境的核心是把隐含前提显式化？
2. 工具链版本漂移为什么会导致“业务没改但项目坏了”的现象？
3. 为什么依赖管理不应只停留在“能 import 成功”？
4. 为什么新人能否快速拉起项目，是环境质量的重要指标？

## 分析题

1. 某项目作者离职后，团队几乎没人能在新机器上完整跑通项目。请分析问题根源。
2. 某 Foundry 项目本地测试全绿，但 CI 经常失败。请分析可能的环境一致性问题。
3. 一个仓库对安装步骤、工具链版本和环境变量几乎没有文档。请分析它在长期维护上的风险。

## 代码题

1. 为一个 Foundry 项目写一份环境初始化清单。
2. 审查你自己的仓库，找出三项“只存在于你脑子里”的环境前提。
3. 写一份新成员入门路径，要求能在最短时间跑通测试和脚本。
