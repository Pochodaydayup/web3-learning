# 第 8 章练习题

## 概念题

1. 为什么说单元测试主要保护的是“你已经想到的行为”？
2. 覆盖率高为什么不等于质量高？
3. 静态分析更适合发现哪一类问题？
4. 为什么成熟的交付流程需要多种工具交叉验证？

## 分析题

1. 某项目 `forge test` 全绿，但权限边界存在遗漏。请分析为什么测试可能没看到这个问题。
2. 某团队把 coverage 从 70% 提到 95%，却仍然频繁出现线上 bug。请分析可能原因。
3. 某仓库接入了静态分析工具，但所有 warning 都被忽略。请分析这和“没接工具”相比有哪些实际差别。

## 代码题

1. 为你的一个合约仓库写一版最小 CI 流程，包含 build、test、coverage 和 static analysis。
2. 审查一个已有测试文件，指出其中三条“执行到了，但断言不够有力”的地方。
3. 为你的项目列一张工具职责表，说明每个命令在防哪类风险。
